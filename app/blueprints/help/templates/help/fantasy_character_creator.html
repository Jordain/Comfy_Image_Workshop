Fantasy Character Creator - Requirements

To be able to use the Fantasy Character Creator, you need to download a custom node from the ComfyUI manager, download a couple models and place it in the apporpriate directory.

  1. Install ComfyUI Essentials from the ComfyUI manager. The ID is 233 and author is cubiq.
  2. Download Proteus V0.3 and place it in ComfyUI_windows_portable\ComfyUI\models\checkpoints.
  3. Download DreamShaperTurbo v2.x and place it in ComfyUI_windows_portable\ComfyUI\models\checkpoints.
  4. Download SDXL LCM Lora then rename to lcm-lora-sdxl.safetensors and place it in ComfyUI_windows_portable\ComfyUI\models\loras.
