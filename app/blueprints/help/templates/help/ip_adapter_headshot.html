Dress U Up - Requirements

To be able to use the Dress U up, you need to download a custom node from the ComfyUI manager, download a couple models and place it in the apporpriate directory and install FaceID dependencies.

  1. Install ComfyUI-IPAdapter_plus from the ComfyUI manager. The ID is 98 and author is cubiq.
  2. Follow the installation instructions from the IPAdapter Plus repo.
  3. Follow the video instructions on installing FaceID
  4. Skip step 3 if you have python 3.11.* installed - download this whl and keep in downloads folder.
  5. Change jorda to your username. Then run this command in cmd within your root ComfyUI directory:

    .\python_embeded\python.exe -m pip install "C:\Users\jorda\Downloads\insightface-0.7.3-cp311-cp311-win_amd64.whl" onnxruntime

  6. Search for #Edit1 in app\app.py then update the app.config. Replace jorda with your username.
  7. Download the model realisticVisionV60B1_v60B1VAE.safetensors and place it in ComfyUI_windows_portable\ComfyUI\models\checkpoints.
