Dress U Up - Requirements

To be able to use the Dress U up, you need to download a custom node from the ComfyUI manager, download a couple models and place it in the apporpriate directory and install FaceID dependencies.

Video Tutorial

Text Instructions
  1. Install ComfyUI-IPAdapter_plus from the ComfyUI manager. The ID is 98 and author is cubiq.
  2. Download the model realisticVisionV60B1_v60B1VAE.safetensors and place it in ComfyUI_windows_portable_CIW\ComfyUI\models\checkpoints.
  3. Download the model CLIP-ViT-H-14-laion2B-s32B-b79K.safetensors download and rename to CLIP-ViT-H-14-laion2B-s32B-b79K.safetensors then place in ComfyUI_windows_portable_CIW\ComfyUI\models\clip_vision.
  4. Download the model CLIP-ViT-bigG-14-laion2B-39B-b160k.safetensors download and rename to CLIP-ViT-bigG-14-laion2B-39B-b160k.safetensors then place in ComfyUI_windows_portable_CIW\ComfyUI\models\clip_vision.
  5. Download the model ip-adapter-full-face_sd15.safetensors and place it in ComfyUI_windows_portable_CIW\ComfyUI\models\ipadapter.
  6. Download the model ip-adapter-faceid-plusv2_sd15.bin and place it in ComfyUI_windows_portable_CIW\ComfyUI\models\ipadapter.
  7. Download the model Ip-adapter-faceid-plusv2_sd15_lora.safetensors and place it in ComfyUI_windows_portable_CIW\ComfyUI\models\loras.
  8. Follow the video instructions on installing FaceID
  9. Skip step 3 if you have python 3.11.* installed - download this whl and keep in downloads folder.
  10. Change jorda to your username. Then run this command in cmd within your root ComfyUI directory:

    .\python_embeded\python.exe -m pip install "C:\Users\jorda\Downloads\insightface-0.7.3-cp311-cp311-win_amd64.whl" onnxruntime

  11. Search for #Edit1 in app\app.py then update the app.config. Replace jorda with your username.
